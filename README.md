# Rajverdhan Vibe Portfolio

Single-page 3D designer portfolio built with:
- React
- Tailwind CSS
- Framer Motion

## Run

```bash
npm install
npm run dev
```

Then open: `http://localhost:5173`

## Key Files

- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/index.css`
- `src/assets/avatar.png`
- `tailwind.config.js`
- `postcss.config.js`
- `vite.config.js`

## Design Notes

- Deep black cinematic theme with neon purple/pink accents and burgundy (`#6d001a`)
- Hero with 3D avatar and glowing gradient CTA
- Framer Motion section reveal animations (fade + slide)
- Numbered services list with glassmorphism hover
- Horizontal project cards with hover scale-up
- Testimonials with outlined cards, mini avatars, and star ratings
- Contact split layout with form + large email link
